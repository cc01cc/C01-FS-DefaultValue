# README

> v0.04

## 功能概述

1. 支持「文本，数字，电话号，单选，多选」类型字段默认值填充
2. 支持单个字段空白记录一键填充，新增记录自动填充
3. 支持多个字段空白记录批量填充，新增记录批量自动填充
4. 支持动态增减字段数量，动态增减字段选项数量
5. 支持本地缓存，刷新插件会自动加载本地缓存
